# Dive Into Deep Learning

<https://d2l.ai/>

## Installation

安装 Python

```shell
conda create --name d2l python=3.9 -y
conda activate d2l
pip install -r requirements.txt
```

下载学习资源

```
curl https://d2l.ai/d2l-en-1.0.3.zip -o d2l-en.zip
unzip d2l-en.zip
cd pytorch
```

打开 jupyter 笔记本

```shell
jupyter lab
```
